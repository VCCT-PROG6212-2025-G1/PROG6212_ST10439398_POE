# Contract Monthly Claim System (CMCS)

## Setup Instructions

### Prerequisites
- Visual Studio 2022
- .NET 8.0 SDK
- SQL Server LocalDB

### Installation Steps

1. **Clone the repository**
```bash
   git clone https://github.com/VCCT-PROG6212-2025-G1/PROG6212_ST10439398_POE
   cd CMCS
```

2. **Restore NuGet packages**
```bash
   dotnet restore
```

3. **Update database connection string**
   - Open `appsettings.json`
   - Verify connection string points to your SQL Server instance

4. **Create database**
   - The database will be created automatically on first run
   - Or run: `dotnet ef database update`

5. **Run the application**
```bash
   dotnet run
```

6. **Login with test accounts**

   **Note:** Click any credential card on the login page for instant auto-fill!

## Features
- ✅ Role-based authentication (Lecturer, Coordinator, Manager, HR)
- ✅ Claims submission with file upload and encryption
- ✅ Multi-level approval workflow
- ✅ Real-time dashboard statistics
- ✅ PDF report generation (invoices, claims reports)
- ✅ RESTful API for automation
- ✅ Swagger documentation (available at `/swagger`)
